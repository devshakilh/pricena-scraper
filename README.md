# Pricena Scraper

A web scraper designed to extract product information from the [Pricena](https://www.pricena.com) website, a price comparison platform.

## Features

- Scrapes product information including:
  - Product name
  - Price
  - Other relevant details
- Provides a concise and structured summary of the scraped data.

## Technology Stack

- **JavaScript**
- **TypeScript**
- **ESLint** for code linting
- **Prettier** for code formatting
- **Husky** for Git hooks

## Installation

To install the necessary dependencies, run:

```bash
npm install
```

## Usage

To start the scraper, use the following command:

```bash
npm start
```

## Configuration

You can configure the scraper settings in the `config.js` file to customize the scraping process.
